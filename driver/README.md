# Flexiband driver
This folder contains basic driver utilities for the flexiband ecosystem.
Currently there are 3 different operating systems supported:

- Microsoft Windows 7 32/64bit (legacy)
- Linux
- Apple macOS

### Linux
To build the linux drivers, use the provided Makefile at `unix/src/Makefile`.

The installation can be accomplished by running the install script located at `unix/install.sh`.

### macOS
Currently only building the drivers for macOS is supported.
To build the drivers for macOS, use the provided Makefile at `unix/src/Makefile`.
